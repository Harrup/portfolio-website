Browser Game Project

View Project View code

The RGB color guess game runs in the browser and is built using HTML, CSS and javascript. Javascript is used to manipulate the Document Object Model (DOM) and to provide the underlying rules for the operation of the game.

Random numbers are generated to create a RGB color that a user must guess correctly in order to succeed at the game.
