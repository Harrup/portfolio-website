Craigslist Clone Project

The craigslist clone project is a redesign of the existing craigslist web application. Built using semantic HTML5 elements such as header, footer, nav and section tags as well as modern CSS3 features like flexboxes, css grid and css variables.

The client side is created using the react javascript framework that makes requests to the backend server in order to acquire data for the user. React router is setup to navigate through the different pages of the application. The backend server is built using Node.js and express as the server side framework. The Backend is designed to filter data before being sent to the frontend as per the users preferences. The web application has a responsive layout achieved through implementation of media queries.
