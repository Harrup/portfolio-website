Mohos Eatery Project

Mohos Eatery is a single page restaurant web application built using the modern Hyperapp javascript library. The web application has dynamic functionality where jsx is used to create a review slider and a special menu that obtain data from the state.

CSS is preprocessed using sass and the applicaton has a responsive layout achieved using the implementation of media queries.
