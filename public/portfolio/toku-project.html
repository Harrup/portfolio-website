Toku Project

Toku is dynamic web application where users can login and share posts with other toku users. It utilizes a combination of client side and server side technologies to authenticate a user, retrieve old posts and save new posts. The client side is built using the react javascript framework with a combination of HTML5 and CSS3, preprocessed using sass. The backend is built using Node.js, AdonisJs server side framework and MySQL relational database.

AdonisJs makes user authentication and validation simple using a controller that contains login, logout and register methods. These methods are triggered when a post request is made from the login and register forms. When a user has successfully logged in, React acquires and displays the users recent posts by setting off a method that fetches the users posts from the database.
